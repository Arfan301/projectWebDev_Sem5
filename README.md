# projectWebDev_Sem5

lokasi folder: xampp/htdocs

git clone https://github.com/Arfan301/projectWebDev_Sem5.git

Lakukan ini ketika pertama kali menclone repository:
- composer install
- npm install
- php artisan migrate
- php artisan key:generate

Untuk meng-akses website :
- nyalakan xampp
- php artisan serve
- npm run dev
